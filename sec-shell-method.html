    Section 3.3 Shell Method

    Exercise 32. Unrolling a Cylindrical Shell.

    A single cylindrical shell has radius \(r\text{,}\) height \(h\text{,}\) and thickness \(\Delta x\text{.}\) When the shell is “cut and unrolled,” the resulting shape is approximately a rectangular solid. What is the approximate volume of that rectangular solid?
    🔗
    • \(2\pi r h\,\Delta x\)
    • Correct — cutting the shell and laying it flat gives a rectangle with length \(2\pi r\) (the circumference), height \(h\text{,}\) and depth \(\Delta x\text{,}\) so \(V \approx 2\pi r h\,\Delta x\text{.}\)
    • \(\pi r^2 h\)
    • This is the volume of a solid cylinder, not a thin cylindrical shell. The shell has a small thickness \(\Delta x\text{,}\) which must appear in the volume expression.
    • \(\pi r h\,\Delta x\)
    • Close, but the circumference of the shell is \(2\pi r\text{,}\) not \(\pi r\text{.}\) When you unroll the shell, the length of the resulting rectangle is the full circumference.
    • \(2\pi r^2\,\Delta x\)
    • This expression is missing the height \(h\text{.}\) The unrolled shell is a rectangular solid whose dimensions are \(2\pi r\) (length), \(h\) (height), and \(\Delta x\) (depth).
    🔗

    Exercise 33. Shell Method Formula Components.

      Match each component in the Shell Method formula, \(V = 2\pi\displaystyle\int_a^b r(x)\,h(x)\, dx\text{,}\) with its geometric meaning.
      🔗
      In the Shell Method, \(r(x)\) is always the distance from the sample shell to the axis of rotation, and \(h(x)\) is the height of the shell (determined by the bounding curves). The factor \(2\pi\) comes from unrolling the shell into a rectangle of length \(2\pi r\text{.}\)
    • \(r(x)\)
    • Distance from the axis of rotation to the shell
    • \(h(x)\)
    • Height of the cylindrical shell
    • \(2\pi\)
    • Comes from the circumference of the shell
    • \(dx\)
    • Thickness of the cylindrical shell
    • \(\pi\)

    Exercise 35. Setting Up a Shell Integral.

    The region bounded by \(y = \sqrt{x}\text{,}\) \(y = 0\text{,}\) \(x = 0\text{,}\) and \(x = 4\) is revolved about the \(y\)-axis. Using the Shell Method, which integral gives the volume of the resulting solid?
    🔗
    • \(\displaystyle 2\pi\int_0^4 x\sqrt{x}\, dx\)
    • Correct — rotating about the \(y\)-axis means \(r(x) = x\text{.}\) The shell height is the vertical distance from \(y = 0\) to \(y = \sqrt{x}\text{,}\) so \(h(x) = \sqrt{x}\text{.}\) Therefore \(V = 2\pi\int_0^4 x\cdot\sqrt{x}\, dx\text{.}\)
    • \(\displaystyle 2\pi\int_0^4 \sqrt{x}\, dx\)
    • This is missing the radius factor \(r(x) = x\text{.}\) The Shell Method integrand is \(r(x)\cdot h(x)\text{,}\) not just \(h(x)\text{.}\)
    • \(\displaystyle \pi\int_0^4 \left(\sqrt{x}\right)^2\, dx\)
    • This is the Disk Method integral (cross-sectional area \(\pi R^2\)), not the Shell Method. The Shell Method uses \(2\pi\int r(x)\,h(x)\, dx\text{.}\)
    • \(\displaystyle 2\pi\int_0^2 y\cdot y^2\, dy\)
    • Switching to \(y\) would be appropriate for the Shell Method only if the axis of rotation were horizontal. Here the axis is the \(y\)-axis (vertical), so we integrate with respect to \(x\text{.}\)
    🔗

    Exercise 36. Horizontal Axis of Rotation.

    When using the Shell Method to revolve a region around a horizontal axis (such as the \(x\)-axis), the radius and height functions must be written in terms of:
    🔗
    • \(y\)
    • Correct — when revolving around a horizontal axis with the Shell Method, the shells are horizontal. Their radius and height are measured as functions of \(y\text{,}\) and the integral is \(2\pi\int r(y)\,h(y)\, dy\text{.}\)
    • \(x\)
    • When the axis of rotation is horizontal, the shells stack vertically, so the natural variable is \(y\text{,}\) not \(x\text{.}\) (Using \(x\) would correspond to the Washer Method for a horizontal axis.)
    • Either \(x\) or \(y\text{;}\) it does not matter.
    • The variable must match the orientation of the shells. For a horizontal axis, shells are horizontal, so \(y\) is the correct variable.
    • A parameter \(t\text{.}\)
    • A parameter is not needed here. The Shell Method uses either \(x\) or \(y\) depending on the axis orientation.

    Exercise 38. Washer vs. Shell Method.

      Match each integral form on the left with the method and axis of rotation on the right.
      🔗
      The Washer Method always squares the radius functions (\(R^2 - r^2\)) and uses \(\pi\text{,}\) while the Shell Method multiplies the radius by the height and uses \(2\pi\text{.}\) The variable of integration tells you whether the method slices perpendicular or parallel to the axis.
    • \(\displaystyle 2\pi\int_a^b r(x)\,h(x)\, dx\)
    • Shell Method — vertical axis
    • \(\displaystyle \pi\int_a^b \Big(R(x)^2 - r(x)^2\Big)\, dx\)
    • Washer Method — horizontal axis
    • \(\displaystyle 2\pi\int_c^d r(y)\,h(y)\, dy\)
    • Shell Method — horizontal axis
    • \(\displaystyle \pi\int_c^d \Big(R(y)^2 - r(y)^2\Big)\, dy\)
    • Washer Method — vertical axis

    Exercise 42. Shell Method: Identify the Error.

    The region bounded by \(y = x^2\) and \(y = 0\) for \(0 \le x \le 2\) is revolved about the line \(x = 3\text{.}\) A student sets up the following integral:
    \begin{equation*} V = 2\pi\int_0^2 x \cdot x^2\,dx = 2\pi\int_0^2 x^3\,dx. \end{equation*}
    Which statement best identifies the error?
    🔗
    • The radius of each shell is the distance from the shell to the axis \(x = 3\text{,}\) which is \(3 - x\text{,}\) not \(x\text{.}\) The correct integral is \(V = 2\pi\displaystyle\int_0^2 (3-x)\cdot x^2\,dx\text{.}\)
    • Correct! When revolving about \(x = c\) with \(c > b\text{,}\) the shell at position \(x\) has radius \(c - x\) (the horizontal distance to the axis). Using \(x\) alone assumes the axis is the \(y\)-axis (\(x = 0\)), which is not the case here.
    • The limits of integration should run from \(0\) to \(3\) to match the position of the axis.
    • Incorrect. The limits of integration always span the \(x\)-extent of the region being revolved, which is \([0,2]\text{.}\) Moving the axis of rotation does not change the limits; it changes the radius expression inside the integrand.
    • The height of each shell should be \(2 - x^2\) rather than \(x^2\text{,}\) because the height is measured downward from \(y = 2\text{.}\)
    • Incorrect. The height of a shell is the vertical extent of the region at a given \(x\text{,}\) which is \(x^2 - 0 = x^2\text{.}\) The value \(y = 2\) has no special role in this problem; the upper boundary is the parabola \(y = x^2\text{,}\) not a horizontal line.
    • The factor \(2\pi\) should be replaced by \(\pi\text{,}\) because the axis of rotation is a vertical line rather than a horizontal one.
    • Incorrect. The factor \(2\pi\) in the shell method represents the circumference of each cylindrical shell and is always present regardless of whether the axis is horizontal or vertical. The orientation of the axis determines the variable of integration and the radius expression, not the leading constant.
