    Exercise 43. Initial Value Problem Solution.

    Which of the following is a solution to the initial value problem
    \begin{equation*} \frac{dy}{dx} = 2y - 10,\quad y(0) = 8\text{?} \end{equation*}
    🔗
    • \(y(x) = 3e^{2x} + 5\)
    • Correct! This function satisfies the differential equation since
      \begin{align*} \frac{d}{dx}\left[3e^{2x} + 5\right] \amp = 2[3e^{2x} + 5]-10 \\ 6e^{2x} \amp = 6e^{2x} + 10 - 10 \\ 6e^{2x} \amp = 6e^{2x} \quad ✅ \end{align*}
      and the initial condition, since \(y(0) = 3(1) + 5 = 8 \quad ✅\text{.}\)
    • \(y(x) = e^{2x} + 5\)
    • This does not satisfy the initial condition since \(y(0) = 1 + 5 = 6 \neq 8 \quad ❌\text{.}\)
    • \(y(x) = 2e^{2x} + 6\)
    • This satisfies the initial condition since \(y(0) = 2(1) + 6 = 8 \quad ✅\text{,}\) but it does not satisfy the differential equation since
      \begin{align*} \frac{d}{dx}\left[2e^{2x} + 6\right] \amp = 2[2e^{x} + 6]-10 \\ 4e^{2x} \amp = 4e^{2x} + 12 - 10 \\ 4e^{2x} \amp = 4e^{2x} + 2 \quad ❌ \text{.} \end{align*}

    • A list of values approximating \(y(t)\) at specific times
    • Correct—numerical solutions are tables of approximate values, not formulas.

    • No, that describes an analytic solution.
    • An exact solution valid for all \(t\)
    • No, numerical solutions are approximations.
    • A graph showing all possible solutions
    • No, that describes a slope field, not a numerical solution.

    Exercise 56. Match Euler’s Method Parts to their Meaning.

      Assume the current approximation using Euler’s Method is denoted by \(y_k\text{.}\) In this context, match the mathematical expression with its meaning.
      🔗
      These pairings are essential when converting a piecewise function into unit step form. The form you choose depends entirely on where each piece is active.
    • \(y_{k+1}\)
    • The \(y\) value of the next approximation.
    • \(y_{k-1}\)
    • The \(y\) value of the previous approximation.
    • \(h\)
    • The distance between \(t_k\) and \(t_{k+1}\text{.}\)
    • \(f(t_k, y_k)\)
    • The slope of the solution at the current approximation.
    • \(t_k\)
    • The \(t\) value of the current approximation.
    • \(h \cdot f(t_k, y_k)\)
    • The rise (or fall) in the \(y\) value from the current approximation to the next approximation.
